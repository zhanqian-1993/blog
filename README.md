# zhanqian/blog

个人技术博客，使用 Astro + Vercel 构建。

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build
```

## 文章写作

在 `src/content/blog/` 目录下添加 Markdown 文件：

```markdown
---
title: '文章标题'
description: '文章描述'
pubDate: '2026-02-09'
tags: ['标签1', '标签2']
---

# 内容
```

## 部署

推送到 GitHub 后，Vercel 会自动部署。

- 网站地址: https://zhanqian-1993.vercel.app
