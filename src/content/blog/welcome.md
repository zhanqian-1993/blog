# 欢迎

这是我的个人博客，记录技术学习笔记和心得体会。

## 如何使用

在 `src/content/blog/` 目录下添加 Markdown 文件即可。

```markdown
---
title: '文章标题'
description: '文章描述'
pubDate: '2026-02-09'
tags: ['标签1', '标签2']
---

# 内容
```

## 技术栈

- **Astro** - 静态站点生成器
- **Vercel** - 免费托管
- **Markdown** - 内容管理
